# Contributing Guidelines / бһӮбҹ„бһӣбһҖбһ¶бһҡбһҺбҹҚбһҺбҹӮбһ“бһ¶бҹҶбһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһ…бһјбһӣбһҡбһҪбһҳ

First off, thank you for considering contributing to the **Khmer Smart Calendar**!  
бһҮбһ¶бһҠбҹҶбһ”бһјбһ„ бһҹбһјбһҳбһўбһҡбһӮбһ»бһҺбһҠбҹӮбһӣбһўбҹ’бһ“бһҖбһ”бһ¶бһ“бһ–бһ·бһ…бһ¶бһҡбһҺбһ¶бһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһ…бһјбһӣбһҡбһҪбһҳбһ…бҹҶбһҺбҹӮбһҖбһўбһ—бһ·бһңбһҢбҹ’бһҚбһ“бҹҚ **бһ”бҹ’бһҡбһҸбһ·бһ‘бһ·бһ“бһҶбҹ’бһӣбһ¶бһҸбһңбҹғбһҒбҹ’бһҳбҹӮбһҡ**!

This document sets the ground rules for contributing to the project.  
бһҜбһҖбһҹбһ¶бһҡбһ“бҹҒбҹҮбһҖбҹҶбһҺбһҸбҹӢбһ…бҹ’бһ”бһ¶бһ”бҹӢбһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһ…бһјбһӣбһҡбһҪбһҳбһҖбҹ’бһ“бһ»бһ„бһӮбһҳбҹ’бһҡбҹ„бһ„бһ“бҹҒбҹҮбҹ”

## 1. Code of Conduct / бһҖбҹ’бһҡбһҳбһҹбһёбһӣбһ’бһҳбҹҢ
By participating in this project, you are expected to uphold a welcoming and respectful environment. Harassment of any kind will not be tolerated.  
бһҠбҹ„бһҷбһҖбһ¶бһҡбһ…бһјбһӣбһҡбһҪбһҳбһҖбҹ’бһ“бһ»бһ„бһӮбһҳбҹ’бһҡбҹ„бһ„бһ“бҹҒбҹҮ бһўбҹ’бһ“бһҖбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһҡбҹҶбһ–бһ№бһ„бһҗбһ¶бһ“бһ№бһ„бһҡбһҖбҹ’бһҹбһ¶бһ”бһҡбһ·бһҷбһ¶бһҖбһ¶бһҹбһҹбҹ’бһңбһ¶бһӮбһҳбһ“бҹҚ бһ“бһ·бһ„бһҖбһ¶бһҡбһӮбҹ„бһҡбһ–бһӮбҹ’бһ“бһ¶бһ‘бҹ…бһңбһ·бһүбһ‘бҹ…бһҳбһҖбҹ” бһҖбһ¶бһҡбһ”бҹҖбһҸбһ”бҹҖбһ“бһӮбҹ’бһҡбһ”бҹӢбһ‘бһҳбҹ’бһҡбһ„бҹӢбһ“бһ№бһ„бһҳбһ·бһ“бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһўбһҸбҹӢбһұбһ“бһІбҹ’бһҷбһЎбһҫбһҷбҹ”

## 2. Reporting Bugs / бһҖбһ¶бһҡбһҡбһ¶бһҷбһҖбһ¶бһҡбһҺбҹҚбһўбҹҶбһ–бһёбһ”бһүбҹ’бһ бһ¶ (Bugs)
If you find a bug, please create a new **Issue** on GitHub. Include:  
бһ”бҹ’бһҡбһҹбһ·бһ“бһ”бһҫбһўбҹ’бһ“бһҖбһҡбһҖбһғбһҫбһүбһ”бһүбҹ’бһ бһ¶бһҺбһ¶бһҳбһҪбһҷ бһҹбһјбһҳбһ”бһ„бҹ’бһҖбһҫбһҸ **Issue** бһҗбҹ’бһҳбһёбһ“бҹ…бһӣбһҫ GitHub бһҠбҹ„бһҷбһ—бҹ’бһҮбһ¶бһ”бҹӢбһҳбһҖбһҮбһ¶бһҳбһҪбһҷбһ“бһјбһңбҹ–
- A clear, descriptive title. (бһ…бҹҶбһҺбһ„бһҮбһҫбһ„бһ…бҹ’бһ”бһ¶бһҹбҹӢбһӣбһ¶бһҹбҹӢ бһ“бһ·бһ„бһ–бһ·бһ–бһҺбҹҢбһ“бһ¶бһ”бһ¶бһ“бһӣбҹ’бһў)
- Steps to reproduce the bug. (бһҮбҹҶбһ бһ¶бһ“бһҠбһҫбһҳбҹ’бһ”бһёбһІбҹ’бһҷбһҷбһҫбһ„бһўбһ¶бһ…бһ”бһ„бҹ’бһҖбһҫбһҸбһҖбҹҶбһ бһ»бһҹбһ“бҹ„бҹҮбһЎбһҫбһ„бһңбһ·бһүбһ”бһ¶бһ“)
- Expected vs. actual behavior. (бһўбҹ’бһңбһёбһҠбҹӮбһӣбһҡбҹҶбһ–бһ№бһ„бһ‘бһ»бһҖ бһ’бҹҖбһ”бһ“бһ№бһ„бһўбҹ’бһңбһёбһҠбҹӮбһӣбһ”бһ¶бһ“бһҖбһҫбһҸбһЎбһҫбһ„бһҮбһ¶бһҖбҹӢбһҹбҹ’бһҸбҹӮбһ„)

## 3. Submitting Pull Requests / бһҖбһ¶бһҡбһҠбһ¶бһҖбҹӢбһҹбҹ’бһ“бһҫбһҹбҹҶбһҺбһҫбһ‘бһ¶бһүбһҷбһҖ (Pull Requests)
We welcome code contributions! Before you submit a PR, ensure you have read our [Development Guide](docs/DEVELOPMENT.md) and [Workflow Guide](docs/WORKFLOW.md).  
бһҷбһҫбһ„бһҹбһјбһҳбһҹбҹ’бһңбһ¶бһӮбһҳбһ“бҹҚбһҡбһ¶бһӣбҹӢбһҖбһ¶бһҡбһ…бһјбһӣбһҡбһҪбһҳбһҹбһҡбһҹбҹҒбһҡбһҖбһјбһҠ! бһҳбһ»бһ“бһ–бҹҒбһӣбһўбҹ’бһ“бһҖбһҠбһ¶бһҖбҹӢбһҹбҹ’бһ“бһҫ PR бһҹбһјбһҳбһ”бҹ’бһҡбһ¶бһҖбһҠбһҗбһ¶бһўбҹ’бһ“бһҖбһ”бһ¶бһ“бһўбһ¶бһ“ [бһҳбһӮбҹ’бһӮбһ»бһ‘бҹ’бһ‘бҹҒбһҹбһҖбҹҚбһўбһ—бһ·бһңбһҢбҹ’бһҚбһ“бҹҚ](docs/DEVELOPMENT.md) бһ“бһ·бһ„ [бһҳбһӮбҹ’бһӮбһ»бһ‘бҹ’бһ‘бҹҒбһҹбһҖбҹҚбһӣбҹҶбһ бһјбһҡбһҖбһ¶бһҡбһ„бһ¶бһҡ](docs/WORKFLOW.md) бһҡбһ”бһҹбҹӢбһҷбһҫбһ„бҹ”

**Checklist (бһ”бһүбҹ’бһҮбһёбһҸбҹ’бһҡбһҪбһҸбһ–бһ·бһ“бһ·бһҸбҹ’бһҷ):**
- [ ] You have created a feature branch (e.g., `feature/awesome-new-thing`). (бһўбҹ’бһ“бһҖбһ”бһ¶бһ“бһ”бһ„бҹ’бһҖбһҫбһҸбһҳбҹӮбһҖбһ’бһ¶бһ„бһҗбҹ’бһҳбһё)
- [ ] Your code follows the existing TypeScript styles and avoids `any`. (бһҖбһјбһҠбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһўбһ“бһ»бһңбһҸбҹ’бһҸбһҸбһ¶бһҳбһҹбҹ’бһҸбһ„бҹӢбһҠбһ¶бһҡ TypeScript)
- [ ] You have written Unit Tests for any new business logic. (бһўбҹ’бһ“бһҖбһ”бһ¶бһ“бһҹбһҡбһҹбҹҒбһҡбһҖбһ¶бһҡбһҹбһ¶бһҖбһӣбҹ’бһ”бһ„бһҜбһҖбһҸбһ¶ (Unit Tests) бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҸбһҖбҹ’бһҖбһңбһ·бһҮбҹ’бһҮбһ¶бһҗбҹ’бһҳбһёбҹ—)
- [ ] All local tests pass (`npm run test:unit` and `npm run test:e2e`). (бһҡбһ¶бһӣбҹӢбһҖбһ¶бһҡбһҹбһ¶бһҖбһӣбҹ’бһ”бһ„бһ”бһ¶бһ“бһҶбҹ’бһӣбһ„бһҖбһ¶бһҸбҹӢбһҠбҹ„бһҷбһҮбҹ„бһӮбһҮбҹҗбһҷ)
- [ ] Your PR targets the **`development`** branch (NOT `main`). (PR бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһ•бҹ’бһҸбҹ„бһҸбһӣбһҫбһҳбҹӮбһҖбһ’бһ¶бһ„ `development` бһҳбһ·бһ“бһҳбҹӮбһ“ `main` бһ‘бҹҒ)

Once you submit your PR, our GitHub Actions CI pipeline will automatically run your code against our testing suite.  
бһ“бҹ…бһ–бҹҒбһӣбһўбҹ’бһ“бһҖбһҠбһ¶бһҖбҹӢбһҹбҹ’бһ“бһҫ PR бһ”бҹҶбһ–бһ„бҹӢ CI бһҡбһ”бһҹбҹӢ GitHub Actions бһ“бһ№бһ„бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһҖбһјбһҠбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһҠбҹ„бһҷбһҹбҹ’бһңбҹҗбһҷбһ”бҹ’бһҡбһңбһҸбҹ’бһҸбһ·бһҠбһҫбһҳбҹ’бһ”бһёбһҹбһ¶бһҖбһӣбҹ’бһ”бһ„бҹ”
